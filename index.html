Beyond Hello World (updated 09/22/2024)

This project uses ruby to generate src code for arrays, dictionaries and classes in 46 different languages.

This project started in 2004 as a way to learn new programming languages. I added new generators as I learned new languages.

As the statistician for our local swim team from 2005 tp 2015, I wrote many scripts in Perl to process and analyze the data from their meets.

Mock Meet Simulator for FSSL 2024(100% Ruby)

I started learning ruby and rewriting many of the scripts in that language. I have programmed with C++, Perl, Java, Python and emacs lisp in my day job as a computer programmer.

After Ruby, I started learning other languages D, Objective C and Smalltalk. I started re writing the scripts in these other languages as well.

Each time I learned a new language, I had to recreate the classes and data structures from scratch in the new language. Instead of repeating this process with every language, I created a Ruby library to generate source code for each new language.

Each language is generated with a ruby module defined for each language
